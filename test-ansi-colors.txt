This is a test file for ANSI Color Viewer extension.

Here are some ANSI color codes to test:

Black text: \033[0;30mThis should be black\033[0m
Red text: \033[0;31mThis should be red\033[0m
Green text: \033[0;32mThis should be green\033[0m
Yellow text: \033[0;33mThis should be yellow\033[0m
Blue text: \033[0;34mThis should be blue\033[0m
Magenta text: \033[0;35mThis should be magenta\033[0m
Cyan text: \033[0;36mThis should be cyan\033[0m
White text: \033[0;37mThis should be white\033[0m

Some variations:
Simple red: \033[31mRed without reset code
Another green: \033[32mGreen text here

Mixed colors in one line:
\033[31mRed\033[0m and \033[32mGreen\033[0m and \033[34mBlue\033[0m

The extension should highlight the ANSI codes themselves (like \033[31m) with the actual colors!
